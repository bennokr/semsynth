Static SPARQL endpoint playground

Endpoint identifier: browser://semsynth-static-catalog. This page embeds YASGUI + the browser Comunica SPARQL engine (no server-side SPARQL service).

The engine queries output/catalog.jsonld and loads query tabs from output/sparql/*.rq files.

SemMap coverage per dataset

Query 1 — Counts SemMap distributions across datasets to confirm semantic metadata coverage.

PREFIX dcat: 
PREFIX dct: 

SELECT ?dataset ?title (COUNT(DISTINCT ?semmap) AS ?semmapCount)
WHERE {
  ?dataset a dcat:Dataset ;
           dct:title ?title ;
           dcat:distribution ?dist .
  ?dist dcat:downloadURL ?semmap .
  FILTER(CONTAINS(LCASE(STR(?semmap)), "dataset.semmap"))
}
GROUP BY ?dataset ?title
HAVING (COUNT(DISTINCT ?semmap) >= 1)
ORDER BY DESC(?semmapCount) ?title

Provenance sources driving synthetic artifacts

Query 2 — Aggregates provenance sources linked to multiple synthetic artifact distributions.

PREFIX dcat: 
PREFIX dct: 
PREFIX prov: 

SELECT ?source (COUNT(DISTINCT ?dataset) AS ?datasetCount) (COUNT(DISTINCT ?artifact) AS ?artifactCount)
WHERE {
  ?dataset a dcat:Dataset ;
           dct:title ?title ;
           prov:wasDerivedFrom ?source ;
           dcat:distribution ?dist .
  ?dist dcat:downloadURL ?artifact .
  FILTER(CONTAINS(LCASE(STR(?artifact)), "synthetic"))
}
GROUP BY ?source
HAVING (COUNT(DISTINCT ?artifact) >= 2)
ORDER BY DESC(?artifactCount) DESC(?datasetCount)

Datasets with SemMap + provenance synthetic outputs

Query 3 — Joins SemMap and provenance-linked synthetic distributions per dataset.

PREFIX dcat: 
PREFIX dct: 
PREFIX prov: 

SELECT ?dataset ?title ?source
       (COUNT(DISTINCT ?semmap) AS ?semmapFiles)
       (COUNT(DISTINCT ?artifact) AS ?syntheticArtifacts)
WHERE {
  ?dataset a dcat:Dataset ;
           dct:title ?title ;
           prov:wasDerivedFrom ?source ;
           dcat:distribution ?semmapDist, ?artifactDist .
  ?semmapDist dcat:downloadURL ?semmap .
  ?artifactDist dcat:downloadURL ?artifact .
  FILTER(CONTAINS(LCASE(STR(?semmap)), "dataset.semmap"))
  FILTER(CONTAINS(LCASE(STR(?artifact)), "synthetic"))
}
GROUP BY ?dataset ?title ?source
HAVING (COUNT(DISTINCT ?artifact) >= 2)
ORDER BY DESC(?syntheticArtifacts) ?title
